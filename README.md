🚀 Electronic Store Web Application
A full-stack e-commerce web application for electronic products with role-based authentication, admin dashboard, and CRUD APIs. The project is deployed using modern cloud platforms for scalability and performance.
⸻
🌐 Live Demo
Frontend (Vercel): https://vercel.com/afroj-shaiks-projects/react-project-1-1nzt
Backend (Render): https://react-project-1-g5yg.onrender.com
⸻
📌 Features
👤 User Features
User registration & login authentication
Browse electronic products
Add products to cart 🛒
View product details
Responsive UI for all devices
🔐 Admin Features
Secure admin login
Role-based access control (Admin/User)
Add, update, delete products (CRUD operations)
Manage product listings efficiently
⸻
🛠️ Tech Stack
Frontend
React.js
HTML5, CSS3
JavaScript (ES6+)
Axios / Fetch API
Backend
Node.js
Express.js
REST APIs
Database
MongoDB (or mention your DB if different)
